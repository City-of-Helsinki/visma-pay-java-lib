Unofficial Visma Pay Java Library
=

See documentation at https://www.vismapay.com/docs/web_payments/

# Install

#### Example for maven:
```
  <dependencies>
    <dependency> 
      <groupId>org.helsinki</groupId>
      <artifactId>visma-pay-java-lib</artifactId>
      <version>1.0-SNAPSHOT</version>
    </dependency> 
  </dependencies>
```
